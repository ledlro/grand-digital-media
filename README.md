# Grand Digital Media ERP

A modern ERP system built with:

- React
- TypeScript
- Vite
- Firebase
- Tailwind CSS

## Installation

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Deployment

This project is deployed using Vercel and uses Firebase for backend services.
